# Push Bot Discord Lên GitHub

Sau khi hoàn thành sửa code bot Discord ThanThu, **luôn push lên GitHub** để deploy.

## Các bước

// turbo-all

1. Kiểm tra thay đổi:
```
git status
```

2. Add tất cả file đã thay đổi:
```
git add -A
```

3. Commit với message mô tả ngắn gọn thay đổi:
```
git commit -m "<mô tả thay đổi>"
```

4. Push lên GitHub:
```
git push
```
