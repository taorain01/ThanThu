# Bot Voice - Đại Ngỗng TTS

Bot Discord chuyên voice TTS (Text-to-Speech) tiếng Việt.

## Cài đặt

```bash
npm install
```

## Cấu hình

Tạo file `.env`:
```
TOKEN=your_discord_bot_token
PREFIX=?
```

## Chạy

```bash
npm start
```

## Lệnh

| Lệnh | Mô tả |
|---|---|
| `?join` | Bot vào voice channel |
| `?leave` | Bot rời voice channel |
| `?stop` | Dừng đọc |
| `.nội dung` | Bot đọc nội dung (tiếng Việt) |
