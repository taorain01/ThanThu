🎲 8. Cơ Chế Roll

🎯 Roll % (Rarity)

Roll ngẫu nhiên 60→100

Tím (60-89)75%
Vàng (90-100)25%

forceGold → roll 90-100 (luôn Vàng)

📋 Roll Loại Dòng

Normal (stat thường)72.5%
Special (dòng phái)17.5%
Universal (Vũ Khí)10%

🎯 Tính Value cụ thể

Normal: value = min + (max - min) × (percent / 100)

Special (Tank): HP×5% + Defense×3% combo

Special (DPS): Attack×5% + Critical×2% combo

Special (Healer): Heal×10% + Affinity×2% combo

Universal: Weapon Attack × 3%

🏆 9. Tính Mastery

Công thức: calculateLineMastery(line, playerClass)

Yếu tốCông thức / Hệ số
Base Points40 + (percent - 60) × 2 → Range: 40 – 120
Hệ số MàuVàng: ×1.25 | Tím: ×1.0
Hệ số Đề CửĐề Cử: ×1.5 | Không: ×1.0
Hệ số PriorityPriority Stat: ×1.3 | Không: ×1.0
Bonus Dòng Đặc BiệtĐúng phái: ×1.3
Bonus Universal×1.15

Ví dụ: Dòng Vàng 95% + Đề Cử + Priority

Base = 40 + (95-60)×2 = 110

→ ×1.25 (vàng) ×1.5 (đề cử) ×1.3 (priority) = 110 × 2.4375 = 268 điểm

Tổng Mastery = Σ mastery tất cả dòng của tất cả 8 slot đang mặc

💬 10. Các Lệnh Tune

LệnhMô tả
?tuneHiển thị Select Menu chọn trang bị Vàng (ưu tiên đề cử cao)
?tune <id>Hiện UI tune chi tiết cho trang bị theo ID
?tune <id> <1-4>Quick tune: tune nhiều dòng cùng lúc
?tune 1-10Bulk tune: tune 1-10 đồ Vàng lên đủ 5 dòng
?tune allTune tất cả đồ Vàng chưa full 5 dòng

🖱️ Interactive UI Features

  • Select Menu: Chọn đồ từ dropdown, ưu tiên dòng Vàng Đề Cử % cao
  • Buttons 1-4 Lần: Tune batch với chi phí hiển thị, disable nếu không đủ đá
  • Button Dòng Cuối: Xuất hiện khi đủ 5 dòng, dùng Thạch Âm
  • Button Gắn Nhanh: Gắn trang bị ngay từ UI tune
  • So sánh: Auto so sánh với đồ đang mặc cùng slot (mastery diff)
  • Perfect Roll: Banner đặc biệt khi roll Vàng + Đề Cử
  • Phù hợp cho: Auto detect class phù hợp nhất theo số dòng đề cử
